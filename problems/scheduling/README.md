# PKI Scheduler

This problem compares different versions of a program for scheduling classes in UNO's PKI building.
The programs were refactored for performance but never formally verified.
This is a work in progress based on this [repo](https://github.com/WhoDis-MaName/Capstone-Team14/tree/anthem-test).

## Usage
```
anthem verify --equivalence external scheduling.1.lp scheduling.2.lp scheduling.ug
```
